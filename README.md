# Gio Portfolio

A clean, responsive personal portfolio showcasing projects, expertise, and certifications.

## Project Structure
```
/
├── assets/                      ← Icons, images, videos, and other assets
├── css/
│   └── style.css                ← Styling and custom layout definitions
├── js/
│   └── script.js                ← Main application logic and interactivity
├── index.html                   ← Portfolio homepage HTML
└── README.md                    ← Project documentation
```

## Features
- **Responsive Layout**: Designed to work beautifully on mobile, tablet, and desktop viewports.
- **Project Showcases**: Video and image project cards with interactive preview behaviors.
- **Skills/Expertise Section**: Dynamically animating skill-level metrics.
- **Certifications Timeline**: Chronological path highlighting professional achievements.
- **Direct Mail Integration**: Easy contact form linked to mail client routing.

## Deployment
This project is built using standard HTML, CSS, and JavaScript with no build steps or dependencies required. It can be hosted directly on static deployment platforms such as:
- **Netlify**: Drop the root folder into the Netlify manual deploy dashboard, or link the Git repository.
- **GitHub Pages**: Enable Pages deployment pointing to the repository's main branch.
- **Vercel**: Deploy as a static project.
